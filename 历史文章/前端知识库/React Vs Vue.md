# React Vs Vue

## 差异简述

Vue 和 React 的核心区别在于设计哲学：Vue 是渐进式框架，追求易用性与开发效率；React 是用于构建用户界面的 JavaScript 库，强调函数式编程与灵活性。‌ 两者在语法、数据流、生态系统和适用场景上均有显著差异。

## 特性区别

React的理念的函数式开发，UI = Function(props)，更加易于扩展和控制；但需要工程师有一定的函数式编程理念以及熟悉jsx的相关内容，学习掌握的难度相对较高。
Vue的理念的基于Proxy/defineProperty实现双向绑定，更接近HTML的模板语言语法，当然也可以通过构建插件支持JSX的写法；更容易从传统的HTML或者初级工程司的入门。
