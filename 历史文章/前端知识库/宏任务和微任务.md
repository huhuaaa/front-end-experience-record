# 宏任务和微任务

## 宏任务

script/setTimeput/setInterval等内部的代码都是宏任务；执行js时第一次是先执行宏任务，再宏任务执行后会先执行微任务队列，直到所有微任务执行完成，再去调度执行宏任务。
所以setTimeout之类的执行哪怕延时时间为0，会在Promise立即执行的代码运行完成之后才能执行，也会在当宏任务同步代码执行之后才执行。

## 微任务

Promise/async/await/queueMicrotask/MutationObserver的相关内容为微任务，比如：

1. Promise.resolve().then(() => console.log('微任务'));
2. await之后的代码，比如：await Promise.resolve(); console.log('微任务');
3. new MutationObserver(() => { console.log('微任务')})

## 特例

IntersectionObserver不能严格算微任务和宏任务，他实际上要等待DOM渲染后才执行。
