题目

有数组 const arr = [-1, 3, 1, 0, 4, 6, 7, 8]，假设长度k=3，k表示在数组中取的数据长度，从index=0开始每次取k个数据，得到数据中的最大值存储起来，得到[3,3,4,6,7,8]。

请实现这个函数，并给出对应的复杂度说明。

---

## 题目分析

窗口滑动过程：

| 窗口范围  | 窗口内元素 | 最大值 |
| --------- | ---------- | ------ |
| [0, 1, 2] | [-1, 3, 1] | 3      |
| [1, 2, 3] | [3, 1, 0]  | 3      |
| [2, 3, 4] | [1, 0, 4]  | 4      |
| [3, 4, 5] | [0, 4, 6]  | 6      |
| [4, 5, 6] | [4, 6, 7]  | 7      |
| [5, 6, 7] | [6, 7, 8]  | 8      |

结果：`[3, 3, 4, 6, 7, 8]` ✓

---

## 解法一：暴力法

```javascript
function maxSlidingWindowBrute(arr, k) {
  const result = [];
  for (let i = 0; i <= arr.length - k; i++) {
    let max = arr[i];
    for (let j = i + 1; j < i + k; j++) {
      if (arr[j] > max) max = arr[j];
    }
    result.push(max);
  }
  return result;
}

const arr = [-1, 3, 1, 0, 4, 6, 7, 8];
console.log(maxSlidingWindowBrute(arr, 3)); // [3, 3, 4, 6, 7, 8]
```

- **时间复杂度**：O(n × k)
- **空间复杂度**：O(1)（不计结果数组）

## 解法二：单调双端队列（推荐）

核心思想：维护一个**递减**的双端队列，队首始终是当前窗口的最大值索引。

```javascript
function maxSlidingWindow(arr, k) {
  const n = arr.length;
  if (n === 0 || k === 0) return [];

  const result = [];
  const deque = []; // 存储索引，对应值单调递减

  for (let i = 0; i < n; i++) {
    // 1. 移除队首超出窗口范围的元素
    if (deque.length > 0 && deque[0] <= i - k) {
      deque.shift();
    }

    // 2. 维护单调递减：从队尾弹出所有 ≤ 当前值的元素
    while (deque.length > 0 && arr[deque[deque.length - 1]] <= arr[i]) {
      deque.pop();
    }

    // 3. 当前索引入队
    deque.push(i);

    // 4. 窗口形成后（i >= k-1），记录结果
    if (i >= k - 1) {
      result.push(arr[deque[0]]);
    }
  }

  return result;
}

const arr = [-1, 3, 1, 0, 4, 6, 7, 8];
console.log(maxSlidingWindow(arr, 3)); // [3, 3, 4, 6, 7, 8]
```

### 执行过程演示

```
i=0, val=-1: deque=[0]                    窗口未形成
i=1, val=3:  弹出0, deque=[1]             窗口未形成
i=2, val=1:  deque=[1,2]                  → 结果: arr[1]=3
i=3, val=0:  弹出2, deque=[1,3]           → 结果: arr[1]=3
i=4, val=4:  弹出3,弹出1, deque=[4]       → 结果: arr[4]=4
i=5, val=6:  弹出4, deque=[5]             → 结果: arr[5]=6
i=6, val=7:  弹出5, deque=[6]             → 结果: arr[6]=7
i=7, val=8:  弹出6, deque=[7]             → 结果: arr[7]=8
```

### 复杂度说明

| 指标                 | 复杂度         | 说明                                  |
| -------------------- | -------------- | ------------------------------------- |
| **时间复杂度** | **O(n)** | 每个元素入队一次、出队一次，均摊 O(1) |
| **空间复杂度** | **O(k)** | 双端队列最多存储 k 个索引             |

## 解法三：数组模拟双端队列（严格 O(1)）

避免 `shift()` 的隐藏 O(k) 开销，用头尾指针实现真正的 O(1) 操作：

```javascript
function maxSlidingWindowOpt(arr, k) {
  const n = arr.length;
  if (n === 0 || k === 0) return [];

  const result = new Array(n - k + 1);
  const deque = new Array(n); // 预分配
  let head = 0, tail = 0;     // head → 队首, tail → 队尾（下一个插入位置）

  for (let i = 0; i < n; i++) {
    // 移除超出窗口的队首
    if (head < tail && deque[head] <= i - k) head++;

    // 维护递减
    while (head < tail && arr[deque[tail - 1]] <= arr[i]) tail--;

    // 入队
    deque[tail++] = i;

    // 记录结果
    if (i >= k - 1) {
      result[i - k + 1] = arr[deque[head]];
    }
  }

  return result;
}

console.log(maxSlidingWindowOpt([-1, 3, 1, 0, 4, 6, 7, 8], 3)); // [3, 3, 4, 6, 7, 8]
```

`head++` 和 `tail--` 都是真正的 O(1)，整体时间复杂度严格 O(n)。
